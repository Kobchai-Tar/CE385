##  Workshop 1.1
ทำงานโดยการประกาศตัวแปรเพื่อเก็บข้อมูลส่วนตัว ได้แก่ ชื่อ อายุ และสาขาวิชา จากนั้นใช้ console.log() แสดงค่าของตัวแปรเหล่านี้ออกทางหน้าจอ โดยมีการเขียนคอมเมนต์เพื่ออธิบายการทำงานของโค้ด
## Workshop 1.2
โปรแกรมสร้างตัวแปรที่เก็บข้อมูลคนละชนิดกัน แล้วใช้คำสั่ง typeof ตรวจสอบชนิดของข้อมูลในแต่ละตัวแปร จากนั้นแสดงผลชนิดข้อมูลออกทาง console เพื่อให้เข้าใจความแตกต่างของ Data Types ใน JavaScript
## Workshop 1.3
โปรแกรมรับค่าคะแนนสอบ 3 วิชา นำมาคำนวณคะแนนรวมและค่าเฉลี่ยด้วยตัวดำเนินการทางคณิตศาสตร์ จากนั้นใช้ตัวดำเนินการเปรียบเทียบเพื่อตรวจสอบว่าค่าเฉลี่ยมีค่ามากกว่าหรือเท่ากับ 80 หรือไม่ แล้วแสดงผลลัพธ์ออกทาง console
## Workshop 1.4
โปรแกรมทำการตรวจสอบข้อมูลผู้ใช้โดยใช้คำสั่งเงื่อนไข if-else และตัวดำเนินการทางตรรกะ เพื่อตัดสินผลลัพธ์การเข้าสู่ระบบตามค่า username, password และอายุ จากนั้นแสดงข้อความที่เหมาะสมตามเงื่อนไขที่ตรวจสอบได้
