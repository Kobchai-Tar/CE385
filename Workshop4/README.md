# Workshop 4.1
Custom Module โดยเขียนฟังก์ชัน operation เพื่อคำนวณทางคณิตศาสตร์ตามประเภทที่กำหนด บวก ลบ คูณ หาร และส่งออกฟังก์ชันนี้ด้วย module.exports จากนั้นนำไปเรียกใช้ในไฟล์ index.js ด้วย require เพื่อทดสอบการทำงาน ฟังก์ชันจะทำงานแบบ Synchronous คือทำทีละคำสั่งตามลำดับและคืนค่าผลลัพธ์ทันที

# Workshop 4.2
การจัดการงานแบบ Asynchronous โดยใช้ setTimeout จำลองงานที่ใช้เวลา
แบบ Callback จะส่งฟังก์ชันเข้าไปเป็นพารามิเตอร์ และจะถูกเรียกเมื่องานเสร็จ
แบบ Promise จะคืนค่าเป็น Promise แล้วจัดการผลลัพธ์ด้วย .then และ .catch ซึ่งอ่านง่ายและจัดการ error ได้ดีกว่า Callback

# Workshop 4.3
เขียน Asynchronous ให้ดูเหมือน Synchronous โดยใช้ async และ await
ฟังก์ชันจะรอผลลัพธ์ของ Promise ก่อนทำงานต่อ และใช้ try และ catch เพื่อดักจับข้อผิดพลาดหากงานล้มเหลวโปรแกรมจะไม่พังและสามารถจัดการ error ได้

# Workshop 4.4
Promise จะทำงาน หลายตัวพร้อมกัน 
Promise.any ใช้เมื่อเราต้องการผลลัพธ์จาก Promise ตัวแรกที่สำเร็จ
Promise.allSettled ใช้เมื่อต้องการทราบผลลัพธ์ของทุก Promise ไม่ว่าจะสำเร็จหรือเกิดข้อผิดพลาด
